# 西南交通大学犀浦校区三维校园导航

本项目是一个基于 Web 的西南交通大学犀浦校区三维可视化导航应用。它旨在通过现代 Web 技术，以三维形式逼真地展示校园环境，并提供精确的地点搜索与路径规划功能。

## ✨ 主要功能

- **三维校园地图**：基于真实的地理数据（OSM），渲染校园内的建筑、道路、水体、绿地等。
- **POI 搜索**：支持校内兴趣点（POI）的模糊搜索与精确定位。
- **路径规划**：提供在校园内的步行路径规划与导航。
- **交互式探索**：用户可以通过缩放、平移、旋转等操作自由探索三维场景。
- **信息面板**：点击建筑或兴趣点可显示详细信息。

## 🚀 快速开始

**环境要求**:
- Node.js >= 16
- pnpm

**安装与启动**:

1.  **进入应用目录**
    ```bash
    cd app
    ```

2.  **安装依赖**
    ```bash
    pnpm install
    ```

3.  **启动开发服务器**
    ```bash
    pnpm run dev
    ```

之后，在浏览器中访问 `http://localhost:5173` 即可看到应用。

更多详细信息，请参阅 [快速入门指南](快速开始.md)。

## 🛠️ 技术栈

- **前端框架**: React 19 + Vite
- **三维渲染**: Three.js + deck.gl
- **状态管理**: Zustand
- **包管理**: pnpm
- **地理数据处理**: osmtogeojson
- **测试**: Vitest, Playwright/Cypress (规划中)

## 📂 项目结构

```
campus-3d/
├── app/                  # Vite + React 应用主目录
│   ├── src/
│   │   ├── components/   # React UI 组件
│   │   ├── three/        # Three.js 场景构建与交互
│   │   ├── deck/         # deck.gl 图层
│   │   ├── store/        # Zustand 状态管理
│   │   ├── lib/          # 核心工具函数
│   │   ├── data/         # 清洗后的 GeoJSON 数据
│   │   └── tests/        # 测试文件
│   └── package.json      # 项目依赖
├── data/                 # 原始数据与中间产物
├── spec/                 # 项目设计与规范文档
├── tools/                # 数据处理脚本
└── map.osm               # 原始 OSM 地图数据
```
